# Student Management System

A web-based Student Management System developed using Python and Flask.  
This project allows managing student records through a simple web interface.

## Features
- Add student details
- View all students
- Search students
- Delete student records

## Technologies Used
- Python
- Flask
- HTML
- CSS
- SQLite
- Git & GitHub

## Project Type
Web-based application

## How to Run the Project
1. Clone the repository
2. Install Flask
3. Run `app.py`
4. Open browser and go to `http://127.0.0.1:5000/`
